# FizzBuzz

## What is this project?

This project is a simple Java program that demonstrates the classic "FizzBuzz" problem, which is commonly used in programming interviews to test basic logic and understanding of loops and conditionals.

The program prints numbers from 1 to 100, but with the following rules:

- If a number is divisible by **3**, it prints **"Fizz"** instead of the number.
- If a number is divisible by **5**, it prints **"Buzz"** instead of the number.
- If a number is divisible by **both 3 and 5**, it prints **"FizzBuzz"**.
- If a number is not divisible by 3 or 5, it simply prints the number itself.

### How it works

The program uses a loop that starts at 1 and ends at 100. For each number, it checks:

1. If the number is divisible by both 3 and 5
2. If the number is divisible only by 3
3. If the number is divisible only by 5
4. Otherwise, print the number

This project helps beginners understand:
- Loops (`for` loops)
- Conditional statements (`if`, `else if`, `else`)
- Modulo operator (`%`) for checking divisibility
